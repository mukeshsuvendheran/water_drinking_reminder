💧 Water Drinking Reminder
A simple Python-based desktop reminder application that notifies users to drink water at regular intervals and logs the activity into an Excel file.

🚀 Features
- ⏰ Hourly desktop notification reminder
- 📊 Logs data into Excel file
- 🕒 Timestamp tracking
- 📁 Auto-creates Excel file if not exists
- 🔁 Runs continuously in background

Project Purpose
This project is a **fundamental building block** for my larger goal of developing:

- Automation systems  
- Continuous monitoring tools  
- Load testing workflows  

It demonstrates:
- Infinite loop execution  
- Time-based triggers  
- Logging mechanisms  
- Real-world automation logic
